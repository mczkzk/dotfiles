# AGENTS.md

## Interaction
- Respond in Japanese.
- Use English for variable names and code comments.
- Be concise.
- Do not start responses with greetings or acknowledgment preambles such as "Understood".
- Bold important parts in responses when it improves scanability.

## Code Style
- Write comments only for intent and rationale, not for obvious operations.
- Match the language of existing documentation and comments in the file you edit.
- When writing dates in documentation, check the current date with `date +%Y-%m-%d` first.
